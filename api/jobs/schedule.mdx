---
title: 'Schedule Job'
api: 'POST /workflows/{workflow_id}/jobs/schedule'
description: 'Manually schedule a job to run immediately for a workflow'
---

<Info>
This endpoint schedules a job with a manual trigger. The job will be queued to run immediately.
</Info>

## Path Parameters

<ParamField path="workflow_id" type="string" required>
  The unique identifier of the workflow
</ParamField>

## Response

Returns HTTP 204 No Content on success.

## Status Codes

<ResponseField name="204" type="No Content">
  Job successfully scheduled
</ResponseField>

<ResponseField name="400" type="Bad Request">
  Invalid workflow ID or missing required parameters
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Authentication required
</ResponseField>

<ResponseField name="404" type="Not Found">
  Workflow not found
</ResponseField>

## Examples

<CodeGroup>
```bash cURL
curl -X POST 'https://api.chronoverse.dev/workflows/wf_abc123/jobs/schedule' \
  -H 'Authorization: Bearer YOUR_API_TOKEN'
```

```javascript JavaScript
const response = await fetch(
  'https://api.chronoverse.dev/workflows/wf_abc123/jobs/schedule',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_TOKEN'
    }
  }
);

if (response.ok) {
  console.log('Job scheduled successfully');
}
```
</CodeGroup>

<Info>
The scheduled job will have trigger type `MANUAL` and will be executed as soon as resources are available.
</Info>
